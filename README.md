# arduino-sound-almabox
Alma box from F.E.A.R. notes for buzzer arduino

Here you can turn on the mini version of the music from the F.E.A.R. - Alma box music.

REMEMBER!

CONNECT **PIN 10** AS PLUS AND **GND** AS MINUS
